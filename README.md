# Permutations-Combinations-Solution

#  Overview

This project is a Python-based interactive tool designed to calculate and visualize key concepts of Permutations and Combinations.
This project focuses on visual learning, helping users understand mathematical concepts through graphs, tree diagrams, and structured representations.

# Features
- Core Concepts
Distinct Permutations → P(n, r)
Distinct Combinations → C(n, r)
Indistinct Permutations 
Combinations with Repetition
- Mathematical Visualizations
  Bar charts for permutations and distributions
Pascal’s Triangle visualization
Tree diagrams for permutations & combinations
Binomial coefficient graphs
Pigeonhole distribution visualization
  - Advanced Topics
Binomial Expansion
Multinomial Expansion
Pigeonhole Principle demonstration

# 🛠️ Technologies Used
Python
Matplotlib (for visualization)
itertools (for combinatorics logic)
math module

# How to Run
Clone the repository:
git clone https://github.com/your-username/permutations-combinations-visualizer.git
Install dependencies:
pip install matplotlib
Run the program:
python main.py
